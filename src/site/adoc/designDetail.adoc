== Design Detail

The following diagram illustrates data flows of the `commandunit`.

[ditaa]
.The Processing Pipeline of the `commandunit`.
----
     You
/-----------\       +-----------+
|   Write   | 1     |{s}        |
|    your   +=----->|   Test    |<----------+
|   Tests   |       |Repository |           |
\-----------/       +-----------+           |
                                            |
                    +-----------+    +------+-----+    +------------+
                    |           |2.1 |{io}        |    |  Compiled  |
                    |           +--->*Preprocessor+=-->|{s} Test    |
                    |           |    |            |    | Repository |
                    |           |    +------------+    +------------+
                    |           |                             ^
                    |           |    +------------+           |
/-----------\ 2     |           |2.2 |{io}        +-----------+
|    Run    +------>*commandunit+--->*  Executor  |
\-----------/       |           |    |            +-----------+
                    |           |    +------------+           :
                    |           |                             V
                    |           |    +------------+    +------------+
                    |           |2.3 |   Report   |    |   Result   |
                    |           +--->*  Renderer  +--->| Repository |
                    |           |    |{io}        |    |{s}         |
                    +-----------+    +------+-----+    +------------+
                                            |
/-----------\       +-----------+           :
|  Examine  | 3     |{d} Test   |           |
|    the    +------>|   Report  |<----------+
|   Report  |       |           |
\-----------/       +-----------+

                                                         +-+    +-+
                                        A writes to B:   |A+=-->|B|
                                                         +-+    +-+

                                                         +-+    +-+
                                        A reads from B:  |A+--->|B|
                                                         +-+    +-+
----
