== Examples


[source, bash]
.Definition of `array_contains` function in `arrays.rc`.
----
function array_contains() {
  local _word="${1}" && shift
  printf '%s\n' "${@}" | grep -q -E '^'"${_word}"'$'
}
----


[source, yaml]
.test-contains_true.yaml++
----
---
"$extends":
  - base/normal.json
when:
  description:
    - Run 'array_contains' with the first argument contained by the rest.
  source:
    - /path/to/arrays.rc
  cmd: array_contains
  args:
    - hello
    - hello
    - world
then:
  exitCode:
    - EQUAL
    - 0
  stdout:
    absent:
      - REGEX:.+
  stderr:
    absent:
      - REGEX:.+
----
